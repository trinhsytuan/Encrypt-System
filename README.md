# Encrypt-System

## Giới thiệu

Encrypt-System là phần mềm mã hoá dữ liệu, tích hợp các thuật toán mã hoá công khai và đối xứng như RSA, Elgamal, A5/1 và tích hợp công cụ chữ ký điện tử như DSS Signature, RSA Signature.

## Tính năng

- Hỗ trợ nhiều thuật toán mã hóa khác nhau (AES, RSA, A5/1, ...)
- Hỗ trợ ghi đè lên file cũ hoặc tạo file mới
- Hỗ trợ lưu lại khoá dưới dạng txt
- Giao diện WinForms trực quan, dễ sử dụng


## Công nghệ sử dụng

- **Ngôn ngữ lập trình:** C#
- **Môi trường phát triển:** Visual Studio 2022


## Cài đặt

### Yêu cầu hệ thống

- Windows 10/11
- Visual Studio 2022
- WinForms
- .NET Framework 4.8 hoặc .NET 6.0

### Cách chạy chương trình

1. Clone repository:

   ```bash
   git clone https://github.com/trinhsytuan/Encrypt-System.git

2. Mở Visual Studio 2022 và load project
3. Build và chạy ứng dụng

## Cách sử dụng

- **Mã hóa:** Chọn thuật toán mã hoá, tạo khoá và chọn file cần mã hoá.
- **Giải mã:** Ngược lại với mã hoá


## Nhược điểm
- Do ưu tiên tới việc dễ cài đặt và lập trình, vì vậy chương trình chỉ hoạt động trên 1 nhân CPU nên tốc độ mã hoá chậm
- Hệ chữ ký điện tử DSS sinh khoá lâu khi ký do tính chất tìm phần tử nguyên thuỷ của hệ mã DSS
